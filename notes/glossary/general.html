Sobre

Aqui você encontrará definições para termos gerais, tipicamente ligados à Filosofia e à Linguística.

C | L | M | P | V

C

Contexto (Linguística).

Contexto é a soma das informações que se fazem necessárias para dar significado a um dado elemento.

L

Linguagem (Linguística).

Uma linguagem é uma forma estruturada de comunicação entre indivíduos de um sistema, sendo formada por um {general:alfabeto} e um vocabulário, tipicamente construído por meio de uma {general:gramática}.

M

Metalinguagem (Linguística).

Uma metalinguagem é uma linguagem utilizada para descrever outras linguagens.

P

Palavra (Linguística).

Uma palavra é a menor unidade de linguagem ao qual está associado um significado dentro de um dado contexto.

V

Vocabulário (Linguística).

Vocabulário é a {general:coleção} de {general:palavras} de uma dada linguagem.
