Here you will find documentation and expositions on some libraries I’m involved.

  1. typed: a Python framework that provides type safety and universal constructions

  2. utils: a collection of typed functions

  3. app: a Python framework to build web applications in a functional approach and presenting type safety

  4. cli: a Python library to quickly construct CLI

Disclaimer. The libraries were constructed mostly to cover my personal needs, and follows my personal point of view on how to build software. They are in constant evolution.

So, use them with caution.

All the libraries are maintained in a separated Github repository. Contributions and suggestions are most welcome! You can create issues directly in the lib repository. For further discussion and to be notified about updates, you are invited to join our Discord.
