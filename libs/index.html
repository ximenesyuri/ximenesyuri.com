Here you will find documentation and expositions on some libraries I’m involved.

  1. typed: a Python framework that provides type safety and universal constructions

  2. utils: a collection of typed functions

  3. comp: a Python component system presenting type safety

  4. cli: a Python library to quickly construct CLI

Disclaimer. The libraries were constructed mostly to cover my personal needs. Also, they follow my personal point of view on how to build and maintain software, and are in constant evolution.

So, use them with caution.

All the libraries are in your corresponding Github repository. Contributions and suggestions are most welcome! You can create issues directly in the lib repository. For further discussion and to be notified about updates, you are invited to join our Discord.
